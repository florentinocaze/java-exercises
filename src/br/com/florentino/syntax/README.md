# Syntax

Exercises focused on Java syntax and core data structures: Strings, arrays, collections, file I/O, methods, and recursion.

---

## Exercises

| # | File | Description | Concepts |
|---|------|-------------|----------|
| 01 | `Ex01StringManipulation.java` | Read a sentence and display: length, uppercase version, reversed version, and vowel count. | String, charAt, length, StringBuilder |
| 02 | `Ex02BasicArrays.java` | Fill an array of 10 integers, find the largest, smallest, and calculate the average. | Array, for, comparison |
| 03 | `Ex03ManualArraySort.java` | Sort an array of integers in ascending order without using Arrays.sort(). | Bubble sort, arrays, nested loops |
| 04 | `Ex04ArrayListUsage.java` | Manage a list of names: add, remove by index, and search by name. | ArrayList, List, collection methods |
| 05 | `Ex05HashMapUsage.java` | Build a dictionary of countries and capitals. Allow lookup by country. | HashMap, Map, get, put, containsKey |
| 06 | `Ex06ExceptionHandling.java` | Prompt for an integer and handle invalid input with a friendly message. | try/catch, InputMismatchException |
| 07 | `Ex07ReadFromFile.java` | Read a .txt file line by line and display its content on the console. | FileReader, BufferedReader, IOException |
| 08 | `Ex08WriteToFile.java` | Receive sentences from the user and save them to a .txt file until "exit" is typed. | FileWriter, BufferedWriter, loop |
| 09 | `Ex09MethodsWithReturn.java` | Refactor the BMI and average exercises using methods with parameters and return values. | Methods, parameters, return, static |
| 10 | `Ex10SimpleRecursion.java` | Calculate factorial and Fibonacci recursively. | Recursion, method calling itself |

---

## Notes

- All exercises read input from the console via `Scanner`;
- File-related exercises (07 and 08) use a local `.txt` file for input/output.
