# Logic

Exercises focused on programming logic fundamentals: operators, conditionals, loops, and real-world formulas.

---

## Exercises

| # | File | Description | Concepts |
|---|------|-------------|----------|
| 01 | `Ex01SumAndComparison.java` | Read A, B, C. Print A+B and whether the result is less than C. | Arithmetic and relational operators |
| 02 | `Ex02EvenOddPositiveNegative.java` | Read a number. Print whether it is even/odd and positive/negative. | `if/else`, modulo operator |
| 03 | `Ex03SumOrMultiply.java` | Read A and B. Sum them if equal, multiply otherwise. Print C. | Nested `if/else` |
| 04 | `Ex04PredecessorSuccessor.java` | Read an integer. Print its predecessor and successor. | Basic arithmetic |
| 05 | `Ex05MinimumWages.java` | Read a salary. Calculate how many minimum wages it represents. | Division, constants |
| 06 | `Ex06SalaryAdjustment.java` | Read a value and print it with a 5% raise. | Percentage, `double` |
| 07 | `Ex07LogicalOperators.java` | Read two booleans. Determine if both are true or both are false. | `&&`, `\|\|` operators |
| 08 | `Ex08DescendingOrder.java` | Read three integers and print them in descending order. | Nested conditionals |
| 09 | `Ex09BMICalculator.java` | Read weight and height. Calculate BMI and print the corresponding condition. | `if/else if`, formula |
| 10 | `Ex10AverageThreeGrades.java` | Read three grades and print the average. | Arithmetic mean |
| 11 | `Ex11StudentApproval.java` | Read four grades, calculate the average and print pass/fail. | Conditionals, average |
| 12 | `Ex12PaymentConditions.java` | Read a product price and payment method. Calculate the final value. | `switch/case`, percentage |
| 13 | `Ex13LegalAge.java` | Read name and age. Print whether the person is of legal age. | Simple conditional |
| 14 | `Ex14SwapValues.java` | Swap the values of A and B without a temporary variable. | Variable manipulation |
| 15 | `Ex15AgeInYearsMonthsDays.java` | Read birth year. Print years, months, and days lived. | Integer division, modulo |
| 16 | `Ex16TriangleType.java` | Read three sides. Validate the triangle and determine its type. | Conditionals, triangle inequality |
| 17 | `Ex17FahrenheitToCelsius.java` | Read a temperature in Fahrenheit and convert it to Celsius. | Conversion formula, `double` |
| 18 | `Ex18FranciscoAndSara.java` | Calculate how many years until Francisco is taller than Sara. | `while` loop, comparison |
| 19 | `Ex19FullMultiplicationTable.java` | Print the multiplication table from 1 to 10. | Nested `for` loop |
| 20 | `Ex20CustomMultiplicationTable.java` | Read a number and print its multiplication table. | `for` loop |
| 21 | `Ex21QuotientAndRemainder.java` | Read A and B. Print the quotient and remainder of the integer division. | Integer division, modulo |
| 22 | `Ex22TeacherNetSalary.java` | Calculate a teacher's net salary given hourly rate, hours, and INSS discount. | Operators, user input |
| 23 | `Ex23FuelConsumption.java` | Calculate fuel used in a trip given time and average speed. | Chained formulas, `double` |

---

## Notes

- Minimum wage reference (Ex05): R$ 1,293.20;
- BMI conditions (Ex09) follow the standard WHO table;
- All exercises read input from the console via `Scanner`.
