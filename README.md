# ☕ Java Exercises

A structured collection of Java exercises organized by topic, built to practice and demonstrate core programming skills – from basic logic to object-oriented design.

---

## 📁 Structure

```
java-exercises/
│
├── logic/       # Operators, conditionals, loops, and math formulas
├── syntax/      # Strings, arrays, collections, files, and methods
└── oop/         # Classes, inheritance, polymorphism, interfaces, and more
```

---

## 📚 Modules

### [Logic](./logic/)
23 exercises covering programming fundamentals: arithmetic and logical operators, conditionals, loops, and real-world formulas (BMI, salary, temperature conversion).

### [Syntax](./syntax/)
10 exercises focused on Java syntax and data structures: String manipulation, arrays, `ArrayList`, `HashMap`, file I/O, exception handling, methods, and recursion.

### [OOP](./oop/)
10 exercises covering object-oriented programming: classes, encapsulation, inheritance, polymorphism, abstract classes, interfaces, enums, and a final mini-system integrating everything.

---

## 🛠️ Technologies

- Java 17+;
- No external dependencies – pure Java SE.

---

## ▶️ How to run

Each file contains a `main` method and can be compiled and run independently.

```bash
# Compile
javac logic/Ex01SumAndComparison.java

# Run
java -cp logic Ex01SumAndComparison
```

Or open the project in any IDE (IntelliJ IDEA, Eclipse, VS Code with Java Extension Pack).

---

## 📝 Conventions

### File naming
- **logic/** and **syntax/**: single-file exercises following the pattern `Ex[number][DescriptionInCamelCase].java`;
- **oop/**: each exercise is a package containing multiple classes and a `Main.java` entry point.
